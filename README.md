# 🎓 SmartClass

Uma plataforma de ensino online desenvolvida para facilitar o acesso a cursos, treinamentos e certificados digitais.

O **SmartClass** oferece uma experiência moderna, intuitiva e organizada, permitindo que estudantes encontrem conteúdos educacionais, acompanhem seu progresso e gerenciem sua jornada de aprendizado de forma simples e eficiente.

---

## 📌 Objetivo do Projeto

Desenvolver uma plataforma educacional capaz de conectar estudantes a conteúdos de qualidade, proporcionando uma experiência de aprendizagem acessível, organizada e moderna.

---

## ✨ Funcionalidades

### 👤 Usuário

* Realizar login na plataforma.
* Visualizar cursos disponíveis.
* Pesquisar cursos por nome ou categoria.
* Acompanhar o progresso de aprendizagem.
* Editar informações do perfil.
* Visualizar certificados conquistados.
* Compartilhar certificados profissionais.

### ⚙️ Sistema

* Gerenciamento de usuários.
* Organização de cursos por categorias.
* Armazenamento de certificados digitais.
* Controle e segurança dos dados.
* Interface responsiva e intuitiva.

---

## 🖼️ Telas do Sistema

## 🚀 Como Executar o Projeto

### Clonar o Repositório

```bash
git clone https://github.com/seu-usuario/smartclass.git
```

### Entrar na Pasta

```bash
cd smartclass
```

### Instalar Dependências

```bash
npm install
```

### Executar o Projeto

```bash
npm run dev
```

A aplicação estará disponível em:

```bash
http://localhost:3000
```
